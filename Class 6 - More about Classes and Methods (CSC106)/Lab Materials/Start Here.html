Before you begin

This is a Xamarin cross-platform class about C#. You can use either Visual Studio on a Mac or Visual Studio on a Windows. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

In this course, you will see a few techniques to make your methods more convenient to use. You will then add constructors to your classes to make it easier to load data into your objects when they are created.

Objectives

  1. Work with optional and named parameters
  2. Using out parameters
  3. Explain how to initialize objects using constructors

Lecture

Learning C# with Xamarin: More about classes and methods

Exercises

  1. Using optional parameters
    Exercise
    In this exercise, you will modify a Xamarin.Forms application to display information about computer purchases. The price you show will be determined by the parameters you pass into a method.
  2. Create a method that returns multiple values
    Exercise
    Here, you will modify a Xamarin.Forms application to get bank account information that includes the account balance and last deposit. Your method will use out parameters to send back data and return a boolean that indicates whether the account is active.
  3. Adding constructors to your class
    Exercise
    In this final exercise, you will add multiple constructors to a BankAccount class to initialize it with different values.

Homework

Homework
This optional homework exercise will help to cement the topics in this class by having you build a simple bank simulation that supports deposits and withdrawals.
