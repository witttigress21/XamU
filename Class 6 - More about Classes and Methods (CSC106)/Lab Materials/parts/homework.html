Learning C# with Xamarin: More about classes and methods

Duration

30 minutes

Homework goals

Here you will write a Xamarin.Forms program to transfer money from your wallet to your bank account. Sample output from the completed application looks like this, notice the "My Wallet" and "Bank Account" labels updating as you deposit and withdraw money.

You will need to perform the following steps:

  1. Create the user interface by adding an Entry, two Buttons, and four Labels.
  2. Use a StackLayout to organize the content.
  3. Build a BankAccount class to manage depositing and withdrawing.

    Hint: The BankAccount class should have a method to deposit, a method to withdraw, and a read only auto property to store the amount of money in the account. You can optionally use a constructor to set the initial amount of money.

  4. Create two private BankAccount fields for your wallet and your bank account in the ContentPage.
  5. Instantiate your BankAccount fields.
  6. Add an event handler to the Clicked event on the deposit and withdraw buttons.
  7. Each time the event handler is called, invoke the appropriate methods on your BankAccount instances.

    Hint: Make sure users can't withdraw money that they don't have. Maybe consider using a return type on your Withdraw method.

  8. Update your Labels to display the correct values.

Solution

The provided Homework Resources/Completed folder contains a completed version of the program you can use to check your work. Your solution does not need to look exactly like this code, however the output should be correct as shown above.
